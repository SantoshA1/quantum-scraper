## Security note

The internal QTP proxy token (48-hex, shared by TSM and both proxies) appears in
this repo by design. Blast radius: paper-only order routing + Telegram sends to
QTP's own chat/channel; the proxy cannot reach the live Alpaca API. ROTATE this
token (and move it out of the repo) before any live-trading credential is created.
Confirmed absent from repo: Alpaca keys, Telegram bot token, GitHub PATs.
